# 🩺 Muhammad Abubakar – Full‑Stack Portfolio

A modern, dynamic portfolio for a **Dispenser & Pharmacy Technician**, built with a Node.js + Express backend and a responsive HTML/CSS/JS frontend. The portfolio features an interactive 3D medical model, a certificate showcase, and an admin panel that lets you update all content in real time.

![Portfolio Preview](https://via.placeholder.com/800x400?text=Portfolio+Preview)

---

## ✨ Features

- **3D Interactive Medical Cross** – Animated with floating particles, rendered with Three.js.
- **Dynamic Content** – All text (name, bio, institute, certificates) is served via a REST API and can be updated through the admin panel.
- **Admin Dashboard** – Click the floating “Admin” button to edit client information and certificate names instantly.
- **Certificate Management** – Four certificate cards with placeholders for images.
- **Fully Responsive** – Works beautifully on desktops, tablets, and mobile devices.
